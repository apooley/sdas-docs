## Overview

SDAS Inventory is the core plugin responsible for managing vehicle data within the SDAS ecosystem. It defines the vehicle custom post type, provides the schema for all vehicle attributes, and powers the frontend display of your inventory. Crucially, it also includes a powerful AI-driven description generator that automatically creates compelling, SEO-friendly vehicle descriptions based on structured data and awards.

## Vehicle Management

The plugin provides a comprehensive interface for managing your vehicle inventory directly within the WordPress admin.

### Vehicle Edit Screen

When adding or editing a vehicle (`Vehicles -> Add New` or editing an existing one), you'll find a structured interface that combines standard WordPress content areas with specific SDAS meta boxes:

-   **Identity & Basic Info**: Fields for VIN, Stock Number, Make, Model, Year, Trim, and Body Style.
-   **Pricing**: Manage MSRP, Internet Price, Book Value, and Invoice pricing.
-   **Mechanical**: Specify Engine Cylinders, Displacement, Drivetrain, Transmission, and Fuel Economy (City/Highway MPG).
-   **Features & Options**: A dedicated area to select vehicle features and options, which are crucial for both frontend filtering and AI description generation.
-   **Media**: Manage the vehicle's image gallery and featured image.
-   **Taxonomies**: Assign the vehicle to specific categories like Condition (e.g., Super Certified, MVI Only), Status, Location, and Colour.

### Bulk Actions

To streamline inventory management, the plugin adds custom bulk actions to the Vehicles list table (`Vehicles -> All Vehicles`):

-   **Generate AI Descriptions**: Select multiple vehicles and trigger the AI description generation process in the background.
-   **Update Status**: Quickly change the status (e.g., Available, Sold, Pending) of multiple vehicles at once.
-   **Regenerate Images**: Trigger a regeneration of image thumbnails for selected vehicles.
-   **Clear Cache**: Manually clear the cache for specific vehicles to ensure frontend updates are immediately visible.

## AI Vehicle Description Generator

One of the standout features of SDAS Inventory is its ability to automatically generate high-quality vehicle descriptions using OpenAI. This feature saves significant time and ensures consistency across your inventory listings.

### How It Works

The generator uses a sophisticated prompt that combines:

1.  **Vehicle Data**: It pulls all available structured data for the vehicle, including Make, Model, Year, Trim, Mileage, Features, and Options.
2.  **Awards Data**: If the SDAS Awards module is active, it injects verified safety ratings (NHTSA, IIHS) and awards (AJAC) into the prompt, ensuring the AI only mentions factual, verified accolades.
3.  **Condition Context**: The prompt adapts based on the vehicle's condition taxonomy (e.g., "Super Certified" vs. "MVI Only"), allowing for tailored messaging.
4.  **Custom Instructions**: It incorporates any custom system prompts or equipment notes you've configured in the settings.

### Generating Descriptions

You can generate descriptions in several ways:

-   **Manually per Vehicle**: On the vehicle edit screen, click the "Generate AI Description" button. The plugin will fetch the data, call the OpenAI API, and inject the resulting text directly into the WordPress editor.
-   **Bulk Generation**: From the Vehicles list table, select multiple vehicles, choose "Generate AI Descriptions" from the Bulk Actions dropdown, and click Apply. This queues the generation process to run in the background via Action Scheduler.
-   **Automatically on Save**: You can configure the plugin to automatically generate a description whenever a vehicle is saved (if it doesn't already have one).
-   **During Import**: The system can be set to automatically generate descriptions for newly imported vehicles.

### Configuration and Settings

The AI description generator is highly configurable via **SDAS -> Inventory Settings -> AI Descriptions**:

-   **Auto-Generate**: Toggle whether descriptions should be automatically generated on save.
-   **Word Count Range**: Set the minimum and maximum desired word count for the generated text.
-   **System Prompts**: Define custom system prompts to guide the AI's tone and style. You can set a default prompt and a specific prompt for "MVI Only" vehicles.
-   **Equipment Notes**: Add specific notes or disclaimers that should be appended to the generated descriptions (e.g., "See dealer for full equipment details").
-   **Generate Missing**: A utility button to quickly queue description generation for all vehicles that currently lack an AI-generated description.

<Callout kind="warning">

**Prerequisites:** The AI description generator requires the SDAS Core plugin to be active and a valid OpenAI API key to be configured within the SDAS Core settings.

</Callout>

## Inventory Settings and Caching

The **SDAS -> Inventory Settings** page also provides controls for managing how inventory data is cached and exported:

-   **Export Caching**: Enable or disable caching for the REST API export endpoints (`/wp-json/sdas/v1/vehicles/export`).
-   **Refresh Frequency**: Configure how often the export cache should be automatically refreshed (e.g., 1-24 times per day).
-   **Manual Cache Clear**: A button to manually purge the export cache, useful after significant inventory updates.

The plugin employs a robust caching strategy, automatically invalidating specific vehicle caches upon save or delete actions, and clearing broader caches after bulk imports are completed.
